# local_dscproctor

This is the first-pass Moodle local plugin for moving the current proctoring system into Moodle-owned PHP.

What this pass includes:

- native Moodle plugin structure
- native Moodle database tables for exams, rooms, attempts, and violations
- admin settings for hybrid migration
- native teacher and student rewrite pages
- optional health check against the legacy backend

Start with the repo-level guide in [MOODLE_NATIVE_REWRITE.md](../../../MOODLE_NATIVE_REWRITE.md).
